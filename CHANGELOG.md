## 1.0.0

- Initial stable release of the Hardcover MCP server
- Improved LLM-oriented tool ergonomics for authenticated user lookups and library status filters
- Improved ambiguous exact-name resolution for author and series lookups, with `_selection.alternatives` when multiple strong candidates exist
- Read tools for books, authors, editions, users, publishers, characters, series, prompts, lists, user libraries, and activity feeds
- Write tools for lists, prompts, prompt answers, user-library entries, and read-history records
